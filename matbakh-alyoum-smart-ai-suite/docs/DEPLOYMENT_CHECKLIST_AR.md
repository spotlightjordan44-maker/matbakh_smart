# قائمة التحقق قبل الإطلاق

1. تشغيل جميع ملفات SQL داخل `supabase/migrations` بالترتيب، بما فيها `0004_knowledge_seed.sql`.
2. تعبئة متغيرات البيئة في Render أو البيئة المستضافة.
3. التأكد من رقم واتساب التجاري والويب هوك.
4. التأكد من أرقام الإدارة في `WA_ADMIN_NUMBERS`.
5. مراجعة مناطق التوصيل ورسومها.
6. مراجعة ساعات العمل والأوقات المتاحة.
7. تفعيل `AI_ENABLED=true` إذا كنت تريد طبقة الفهم الذكي.
8. تفعيل `OPENAI_API_KEY` إذا كنت تريد طبقة OpenAI الاختيارية.
